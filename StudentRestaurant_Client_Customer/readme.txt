1. KIOSK 설정 방법: 메모장으로 kiosk.properties 파일을 수정합니다.
 - kioskName: 키오스크 메인화면과 식권 등에 표시될 식당 이름을 입력합니다.
 - kioskNo: 여러 대의 키오스크를 운영하는 경우 번호를 구분하여 입력합니다. 한 대만 운영하는 경우 1로 설정하시기 바랍니다.
 - serverUrl: 서버(영양사실)측에 문의하여 알아낸 서버 주소를 입력합니다.
 - serverPort: 서버(영양사실)측에 문의하여 알아낸 서버의 포트 번호를 입력합니다.
 
2. KIOSK 메인 화면 종료 방법: 키오스크 화면에서 배너 부분을 클릭한 다음 q 버튼을 눌러 종료합니다.
